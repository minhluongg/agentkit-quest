## The rename

ClaudeKit became **AgentKit**. `claudekit.cc` now redirects to `agentkit.best`, and the CLI moved
from `ck` to `ak`.

## What ships in the box

<Steps>
  <Step>
    ### Install the CLI

    ```bash
    npm install -g agentkit-cli
    ```
  </Step>
  <Step>
    ### Initialize a project

    ```bash
    ak init --kit engineer
    ```
  </Step>
</Steps>
