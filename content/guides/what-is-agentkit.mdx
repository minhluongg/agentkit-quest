**AgentKit** is a paid toolkit for [Claude Code](https://code.claude.com) — a curated,
maintained set of skills, specialised subagents, and hooks, pre-wired into a
plan → build → review → ship workflow. It was previously called **ClaudeKit**; the name
changed in 2026 (see [the migration note](/guides/claudekit-to-agentkit)).

It does not add anything Claude Code technically withholds — skills, subagents and hooks
are all [free, first-party features](/guides/skills-vs-commands-vs-subagents). What you
are buying is the **curation and the wiring**: a large set of pieces that already fit
together, so you are not authoring and tuning dozens of config files yourself.

## What's actually in it

AgentKit ships as two kits (plus a bundle):

| Kit | For | Contains |
|---|---|---|
| **Engineer** ($99) | Developers | 88 skills, 30+ workflows, 13 subagents — frontend, backend, database, DevOps, code review, debugging |
| **Marketing** ($99) | Growth teams | 28 marketing subagents, 12+ MCP integrations — research, SEO, email, CRM |
| **Bundle** ($149) | People who do both | Everything in both |

The Engineer row is counted from the kit itself (v2.20.0). AgentKit's own site advertises
"60+ skills" and "17 agents"; we publish what is actually in the build we installed. The
Marketing figures are the vendor's — we do not own that kit and cannot count it.

This site documents the **Engineer** kit. Its 88 skills fall into categories you can
browse in full on the [skills page](/skills):

- **Development Tools (24)** and **Utilities (32)** — the bulk: planning, cooking
  (implementing), reviewing, testing, debugging, git, scouting a codebase
- **Frontend & Design (12)**, **Backend (3)**, **Database (1)**, **Infra/DevOps (2)**
- **AI/ML (3)**, **Multimedia (5)**, **Frameworks (4)**, **Security (1)**

And **13 subagents** — planner, researcher, code-reviewer, debugger, tester, and more —
each running in [its own context window](/guides/skills-vs-commands-vs-subagents) so they
can do heavy work without flooding your main conversation.

## The three building blocks

Everything in the kit is one of three Claude Code primitives. If you understand these,
you understand what the kit is made of:

- **Skills** — reusable instructions Claude loads when relevant (`/ak:plan`, `/ak:cook`).
- **Subagents** — workers with a separate context that return a summary.
- **Hooks** — scripts that fire automatically on events (before a command runs, after a
  file is written). AgentKit uses these for guardrails.

We wrote a [full guide to the difference](/guides/skills-vs-commands-vs-subagents), and a
[set of 26 free hooks](/guides/claude-code-hooks-examples) you can use with or without the
kit.

cases where you should **not** buy it, and a guide to
